# 01_Datos

Esta carpeta contiene los datos utilizados en el proyecto.

## Contenido

Los datos provienen de fuentes públicas del ICFES (Instituto Colombiano para la Evaluación de la Educación):

- **Fuente:** Portal de datos abiertos del gobierno colombiano (datos.gov.co)
- **Período:** 2019-2024
- **Registros:** 1,217,482 estudiantes
- **Formato:** CSV

## Variables principales

- `mod_razona_cuantitat_punt`: Variable objetivo (0-300 puntos)
- `mod_lectura_critica_punt`: Predictor numérico
- `mod_comuni_escrita_punt`: Predictor numérico
- `mod_competen_ciudada_punt`: Predictor numérico
- `mod_ingles_punt`: Predictor numérico
- Variables sociodemográficas categóricas

## Nota sobre privacidad

Todos los datos son públicos y anonimizados. Cumplen con RGPD y Ley 1581/2012 de Colombia.
